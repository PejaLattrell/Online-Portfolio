# My Website

This project is a complete website that includes both frontend and backend functionality. Below are the details of the project structure and setup instructions.

## Setup Instructions

### Frontend

1. Navigate to the `frontend` directory.
2. Open `index.html` in a web browser to view the website.
3. Modify `main.css` to change styles and `main.js` for interactivity.

### Backend

1. Navigate to the `backend` directory.
2. Install dependencies using `npm install`.
3. Create a `.env` file with your environment variables.
4. Start the server by running `node src/app.js`.

## Features

- Responsive design with CSS.
- Dynamic content handling with JavaScript.
- RESTful API for backend functionality.
- Modular structure for easy maintenance and scalability.
